# 스크립트 실행 방법

## 1. Docker 설치 (install-docker.sh)

```bash
chmod +x install-docker.sh
sudo ./install-docker.sh
```

- Docker Engine, Docker Compose 플러그인, 공유 네트워크(`jumsim_network`)까지 자동 설정
- 실행 후 docker 그룹 적용을 위해 터미널 재접속 또는 `newgrp docker` 실행 필요

## 2. GitHub SSH 연결 (github-ssh-script.sh)

```bash
chmod +x github-ssh-script.sh
./github-ssh-script.sh
```

- SSH 키 생성 → GitHub 공개 키 등록 → 연결 테스트 → Git 사용자 설정 → redis-infra 저장소 클론까지 순서대로 진행
- 스크립트 실행 중 아래 입력이 필요함:
  1. GitHub 이메일 주소
  2. GitHub에 공개 키 등록 후 Enter
  3. Git 사용자 이름
